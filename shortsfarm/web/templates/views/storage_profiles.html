Профили — это локальные каналы

В этой вкладке остаются только карточки профилей и общие настройки. Нажмите профиль, чтобы открыть отдельную страницу канала с видео, очередью публикации и YouTube-синхронизацией.

YouTube подключается отдельно

Сами аккаунты YouTube и Google API auth настраиваются в разделе «Интеграции», а привязка к конкретному профилю — внутри страницы этого профиля.

Профили используют глобальные теги

Теги создаются и редактируются в отдельном разделе «Теги». Профиль только подключает include/exclude правила.
